# che-rewriter

Fine-tuned LLM project for rewriting neutral Spanish into Rioplatense Spanish with control over voseo, register, and lunfardo intensity.

## Goal

Build a portfolio-ready fine-tuning project focused on controlled Spanish localization.

The model should preserve meaning while adapting:

- voseo
- register
- lunfardo intensity
- Rioplatense tone

## Initial stack

- Python
- Hugging Face Transformers
- TRL
- PEFT
- bitsandbytes
- FastAPI
- Pytest
